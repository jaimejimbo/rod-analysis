Rod-analysis

Tools for analyzing granular rod behaviour.

View on GitHub Download .zip Download .tar.gz

This tools are for studying granular rod-like particles's beaviour when situated into a vibrated mono-layer system.

To use it, just download the source, install matplotlib, scipy, numpy and imagej and run main.py. You can change settings in main.py.
